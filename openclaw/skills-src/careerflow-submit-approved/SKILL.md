---
name: careerflow-submit-approved
description: Use the OpenClaw browser to submit an application or send outreach only after explicit approval in the current session.
---

Use this skill only after the user has explicitly approved a prepared application or outreach bundle.

Required behavior:

- Require explicit user approval in the current OpenClaw session.
- Reconfirm the target company and role before the final action.
- Use the reviewed artifact bundle and the live destination form or message draft as the source of truth.
- Prefer preparing the final browser form fill or email draft first unless the user explicitly asks for the final send or submit.
- Summarize exactly what was submitted or sent, including the company, role, destination URL or recipient, and any answers or attachments used.
- If anything in the live form diverges from the approved bundle, stop and return to review rather than improvising.
